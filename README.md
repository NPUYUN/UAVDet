# UAVDet

面向低空复杂场景的无人机目标检测研究项目，围绕“轻量化 + 高精度 + 实时性”开展算法、数据与系统一体化设计。

## 项目简介

随着无人机在巡检、安防、应急等场景中的广泛应用，低空目标监测需求快速增长。无人机目标通常具有**尺度小、背景复杂、运动快、易遮挡**等特点，传统检测方法在实际部署中常面临以下矛盾：

- 检测精度与实时性难以同时满足；
- 轻量化部署与特征表达能力存在冲突；
- 真实数据采集与标注成本高，影响模型泛化；
- 工程系统需要长期稳定运行与可维护性保障。

UAVDet 针对上述问题，构建了从模型优化到系统落地的完整技术方案。

## 研究主线

本项目包含三条核心研究主线：

1. 轻量化高性能检测算法：`RepViT-Drone`
2. 多尺度特征增强检测算法：`HHFNet`
3. 多模型融合检测系统与仿真数据生成支撑

## 方法概览

### 1) RepViT-Drone（基于 YOLOv11-N 的轻量化增强）

为在实时推理约束下兼顾精度与模型规模，RepViT-Drone 在 YOLOv11-N 基础上引入：

- `RepViTBlock`：增强跨区域、跨通道特征交互；
- 通道注意力下采样：在压缩过程中保留关键语义与小目标信息；
- `RIC`（Residual Inverse Convolution）：增强上采样阶段细节恢复；
- `AFCA`（Adaptive Feature Calibration Attention）：多尺度融合后自适应校准关键区域。

论文实验结论显示，RepViT-Drone 在精度、速度与模型体积间实现了更优平衡，且模型体积约在 `1.8MB ~ 1.9MB` 量级，具有较强嵌入式部署潜力。

### 2) HHFNet（轻量化多尺度特征增强网络）

针对单阶段检测在局部多尺度建模不足的问题，HHFNet 重点引入：

- `CSF`（Cross-Scale Feedback）：强化深层语义与浅层细节协同；
- `PRFA`（Progressive Receptive Field Aggregation）：渐进式扩展感受野并增强高层多尺度表达；
- 结构轻量化优化：在保持较高推理效率下提升复杂背景和小目标检测稳定性。

论文结论表明，HHFNet 在复杂背景与多尺度目标场景下具备更强鲁棒性，并在精度、实时性、模型规模之间取得均衡表现。

### 3) 多模型融合检测系统

为提高工程场景下的稳定性与可维护性，系统采用模块化与多线程设计，包含：

- 输入接收与任务调度；
- 单模型推理与融合推理；
- 结果展示与历史回溯；
- 数据管理与仿真数据生成控制。

系统支持两类融合策略：

- `WBF`（Weighted Boxes Fusion）：偏重综合定位质量；
- `SBF`（Selective Box Fusion）：引入选择机制，适合精确度优先场景。

实验结论显示，多模型融合在精度与误检控制方面整体优于单模型方案。

## 数据集与训练设定

### 使用数据集

- `DUT Anti-UAV`：多场景、多光照（城市/森林/农田/天空等），覆盖典型无人机检测难点；
- `Det-Fly`：更高挑战的微小目标场景，用于鲁棒性与泛化能力评估。

### 训练与增强（论文公开信息）

- 输入分辨率统一为 `640×640`；
- 常用增强策略：平移、缩放、随机左右翻转等；
- 检测评估指标：`Precision`、`Recall`、`mAP50`、`mAP50:95`、`Latency`、`FPS`、`Model Size`。

## 主要结论（来自论文）

- RepViT-Drone 在两个无人机数据集上均表现出良好的综合性能，尤其在轻量化与部署友好性方面优势明显；
- HHFNet 在多尺度与复杂背景场景下具有更强检测稳定性，对小目标识别能力更突出；
- 多模型融合系统可进一步提升检测稳定性与可靠性，适配不同任务优先级（精度优先或召回优先）；
- 仿真数据生成系统有效缓解真实数据采集与标注压力，并提升复杂场景下泛化效果。

## 项目创新点

### 算法层

- 基于 RepViT 思路改造 YOLOv11-N，提升轻量网络的特征表达与细节恢复；
- 引入局部多尺度协同机制（CSF + PRFA），强化小目标检测能力。

### 数据层

- 构建面向无人机检测的仿真数据生成流程，增强数据多样性与可控性。

### 系统层

- 设计多线程、模块化、多模型融合检测系统，兼顾实时性与工程可维护性。
